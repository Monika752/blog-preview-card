# Blog-preview-card

Frontend Mentor Challenge #2 – Blog Preview Card built using HTML and CSS. A responsive card component showcasing a blog article with 
clean typography, proper spacing, and modern UI design.

## Overview

This is my solution to the Frontend Mentor **Blog Preview Card** challenge.

The project consists of a responsive blog card featuring an illustration, category badge, publication date, article title, 
description, and author information. The goal was to practice semantic HTML, CSS styling, Flexbox, spacing, typography, shadows, 
and recreating a Figma design with pixel-perfect accuracy.

## Features

- Responsive card design
- Semantic HTML5 structure
- Clean and modern UI
- CSS Variables for reusable styling
- Flexbox for layout and alignment
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Figtree)

## Live Demo

**Link:** https://monika752.github.io/blog-preview-card/

## What I Learned

- Building responsive card components
- Structuring webpages with semantic HTML
- Using CSS Variables for consistent styling
- Applying Flexbox for alignment and layout
- Managing spacing with margin and padding
- Creating modern UI using borders, shadows, and border-radius
- Converting a Figma design into a responsive webpage
